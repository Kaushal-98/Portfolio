# Kaushal Sharma — Portfolio

An interactive developer portfolio built with React, TypeScript, and Framer Motion.

**Live:** [Check out](https://portfolio-kaushal-sharma.vercel.app/)

## Features
- Smooth-scroll single-page layout with a sticky nav bar
- ⌘K command palette
- Working terminal (try `help`)
- Real project case studies, tech stack, and education timeline
- Custom cursor, aurora background, glassmorphism UI

## Tech Stack
React · TypeScript · Vite · Framer Motion · GSAP · Lenis

## Run locally
```bash
npm install
npm run dev
```
